# CRM Alma Orders

Sistema de gestión comercial para Alma Orders.

## Módulos
- Dashboard con métricas en tiempo real
- Profesionales
- Tiendas
- Comercios
- Riders
- Embajadores
- Prospectos (Pipeline Kanban)
- Seguimientos
- Renovaciones
- Estadísticas con gráficos SVG
- Notificaciones
- Buscador Global
- Actividad / Notas
- Configuración

## Deploy
Publicado en Vercel. Base de datos en Supabase.
